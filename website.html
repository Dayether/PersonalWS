Projects

Equitrace

EquiTRACE is a Java program built using JFrame to address SDG 16. It allows users to report issues by selecting a category and typing in their complaints. The program promotes peaceful and inclusive societies, ensures access to justice, and supports the development of effective, accountable institutions.

IT Student Club System

The IT Student Club System is a mobile app built using Java in Android Studio. It helps manage IT student organizations by allowing admins to add, update, and delete member records, post announcements, and respond to member queries. The app provides a simple and user-friendly interface for efficient club management.

PeriodicPal

PeriodicPal is a Java program built using JFrame. It provides an interactive interface for users to access information about elements in the periodic table, including their name, symbol, atomic number, and atomic weight. The program uses file reading, arrays, string manipulation, and multithreading to enhance functionality.
